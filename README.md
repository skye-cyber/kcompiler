# kcompiler
Compile kotlin files to .jar files,
Run .jar files using java,
Compile kotlin directories.

# Requirents
```shell
openjdk-11-jdk
kotlin
```

# Applicable commands
```shell
krun
kcompiler
compile
kotlin_compiler
compile_kotlin
```
# Arguments
```shell
-h/--help help message
```
```shell
-run run only
```
```shell
-CR compile and run
```
```shell
input target file or folder/directory
```

# Usange
```shell
kcompiler hello.kt
```
```shell
kcompiler hello.kt -CR
```
```shell
kcompiler /home/user/kotlin
```
```shell
kcombiler hello.jar -run
```
